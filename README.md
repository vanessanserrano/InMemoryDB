# In-Memory Database

This project creates an in-memory key database that help transaction operations.

Methods:

- `begin_transaction()`
- `put(String key, int value)`
- `Integer get(String key)`
- `commit()`
- `rollback()`

---

## Run Project

### Requirements
- Java 17 +
- Terminal

### Steps

1. Clone or download this repository and navigate into the project folder:

   ```bash
   cd InMemoryDB
2. Compile
    javac src/*.java
3. Run
    java -cp src Main

Creates an instance of the in-memory DB
